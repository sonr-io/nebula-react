# Avatar
Sonr SSO Avatar component.

# Properties
```js
{
    src?: string;                 // for displaying an image
    alt: string;                  // 2 Characters text to display when there is no image src
    size?: 'lg' | 'md' | 'sm';    // to define height and width equivalent to h-12 w-12, 10, and 8 from Tailwind.css v3 (default 'md')
    style?: CSSProperties;        // optional css classes to apply
    nearby?: boolean;             // resposible to display the light blue shadowy border around the Avatar
    color?: 'gray' | 'purple' | 'green' | 'red' | 'orange' | 'teal';  // color options for the background when there is no image (default purple)
}
```
